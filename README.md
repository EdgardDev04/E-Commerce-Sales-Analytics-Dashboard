# E-Commerce-Sales-Analytics-Dashboard
An end-to-end data analytics project designed to extract actionable insights from e-commerce sales data. This dashboard provides key business metrics, customer behavior analysis, and predictive forecasting to support data-driven decision-making.

Key Features:

Monthly and yearly revenue analysis

Top-selling products and categories

Customer lifetime value (CLV)

Churn rate analysis

Cohort analysis

Conversion rate tracking

Sales trend visualization

Time-series forecasting (ARIMA/Prophet)

Anomaly detection in revenue patterns

Tech Stack:

Python (Pandas, NumPy, Scikit-learn)

SQL

Matplotlib / Plotly

Power BI or Streamlit

PostgreSQL

Objective:

Transform raw transactional data into meaningful business intelligence insights and predictive analytics models for strategic planning.
